function os_thread_create

Create a thread and add it to Active Threads and set it to state READY. 

uint32_t os_thread_create (
    void(*)(const void *argument) task,
    void * argument,
    int priority,
    uint32_t stack_size
)

Parameters:

  • task Function pointer which is the thread body. It should not run into the end of function unless os_thread_terminate is invoked
  • argument the data pointer which brings to task
  • priority The underlying os is FreeRTOS. It executes tasks with highest priority which are not in idle state. If there are more than 2 tasks to be executed, then they share the time slice.
  • stack_size The stack_size is used as memory heap only for this task. The local variables and call stacks would occupy this heap. Please make sure the the stack_size is big enough to avoid curroption

Returns:

The thread id which is used in thread operation and signaling.

function os_thread_yield

Pass control to next thread that is in state READY . 

uint32_t os_thread_yield (
    void
)

By default the minimal execution unit is 1 millisecond. In a scenario that if a thread with smaller want to handout execution right to a thread with higher priority immediately without waiting for the ending of current 1 millisecond, then invoke os_thread_yield can transfer exection right to OS's idle task and check which is the next execution thread.

os_status code
